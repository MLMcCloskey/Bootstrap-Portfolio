memePie

memePie is an image sharing application I built with a few friends from Georgia Tech. It is designed as a place for friends to post and share memes they create or find on the internet. It was built entirely with the MERN stack, including a few additional Amazon Web Services (for cloud storage and user authentication) and functions as a Progressive Web App. Currently has active users in 3 continents.
